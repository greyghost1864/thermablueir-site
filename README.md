# ThermaBlue IR Website

Deploy-ready Next.js package for thermablueir.com.

## What is included
- Single-page launch site
- Public-safe positioning for ThermaBlue IR as a manufacturing + infrastructure platform
- Heating & Air as the catalyst product
- Selective Pete integration
- Approved image set wired into `/public/images`

## Local run
1. Install Node.js 20+
2. In this folder, run `npm install`
3. Run `npm run dev`
4. Open `http://localhost:3000`

## GitHub + Vercel deployment
1. Create a free GitHub account
2. Create a new repository named `thermablueir-site`
3. Upload all files in this folder to the repository
4. Create a free Vercel account
5. In Vercel, click **Add New... > Project**
6. Import the GitHub repository
7. Keep the default Next.js settings and deploy
8. In GoDaddy DNS, point `thermablueir.com` to Vercel and set `www` to redirect
